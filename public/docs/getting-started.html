Instalacja

Aby rozpocząć pracę z frameworkiem BubbleCSS należy umieścić natępujące linki w znaczniku <head>.

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page title</title>
    <link rel="stylesheet" href="https://adambru.github.io/BubbleCSS/main.css">
    <script src="https://adambru.github.io/BubbleCSS/script.js" type="module" data-base-path="../" data-logo-path="bubblecss-logo-aqua.png"><script>
</head>

Alternatywnym sposobem korzystania z BubbleCSS jest pobranie paczki BubbleCSS.zip, rozpakowanie jej i umieszczenie folderów w sposób opisany w sekcji Struktura projektu.

Struktura projektu

Aby framework działał poprawnie, musi zostać zachowana właściwa organizacja plików.

Struktura folderów i plików

/ (project root)
├─ assets
│  └─ images
│  └─ config.css
│  └─ style.css
├─ components
├─ index.html
├─ subfolder1
│  └─ subpage1.html
└─ subfolder2
   └─ subpage2.html
