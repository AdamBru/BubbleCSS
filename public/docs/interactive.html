Interaktywne

Przyciski

Przycisk ważna sprawa. Aby go stworzyć wystarczy dodać klasę "btn" do znacznika <button>. Aby zmienić jego kolor, należy dodać do niego klasę odpowiedniego koloru. 

<button class="btn">Guziol</button>
<button class="btn color-main">Kolor główny</button>
<button class="btn color-accent-1">Kolor akcentu 1</button>
<button class="btn color-accent-2">Kolor akcentu 2</button>
<button class="btn color-green">Akceptuj</button>
<button class="btn color-yellow">Uwaga</button>
<button class="btn color-red">Anuluj</button>
<button class="btn color-blue">Informacja</button>

Pola tekstowe

Pole tekstwe to standardowy znacznik <input>. Pole domyślnie rozciągnie się na całą szerokość rodzica, aby je ograniczyć można zastosować style="max-width: ". 

<input style="max-width: 300px;">

Pole tekstowe można wzbogacić o Dodatki do pól tekstowych. Przed dodaniem Dodatku do pól tekstowych należy "obciąć" pożądaną stronę pola tekstowego. 

<input class="input-flat-right">
<input class="input-flat-right">
<input class="input-flat-both">

Pole tekstowe z Dodatkiem do pól tekstowych trzeba ująć w <div> z klasą "input-group".

class="input-addon-right"
class="input-addon-left"
class="input-addon-left" class="input-addon-right"
class="input-addon-center" 
<!-- (input][addon) -->
<div class="input-group">
    <input class="input-flat-right">
    <span class="input-addon-right"></span>
</div>

<!-- (addon][input) -->
<div class="input-group">
    <span class="input-addon-left"></span>
    <input class="input-flat-left">
</div>

<!-- (addon][input][addon) -->
<div class="input-group">
    <span class="input-addon-left"></span>
    <input class="input-flat-both">
    <span class="input-addon-right"></span>
</div>

<!-- (input][addon][input) -->
<div class="input-group">
    <input class="input-flat-right">
    <span class="input-addon-center"></span>
    <input class="input-flat-left">
</div>

Odnośniki

Znacznik <a> to typowy znacznik odnośników w HTML. Można go stylizować jak Przyciski.

Strona główna 
<a href="../index.html" class="btn">Strona główna</a>

Odnośniki mogą również znajdować się w tekście. Należy wówczas użyć klasy "link".

Lorem ipsum dolor sit amet.

<p>Lorem <a href="#" class="link">ipsum</a> dolor sit amet.</p>

Tooltip

Tooltip, czyli niewielki dymek nad elementem można dołożyć do dowolnego innego elementu. Naleleży do owego elementu dodać klasę "tooltip" oraz atrybut data-tooltip="" z treścią dymku wpisaną w cudzysłowie. 

<button class="btn tooltip" data-tooltip="Jestem tooltipem">Najedź na mnie</button>

<div class="input-group tooltip" data-tooltip="Naleleży wypełnić to pole">
    <input type="text">
</div>

Karty

Modal

Wyskakujące okno, modal, można wywołać funkcją dodaną do przycisku, w nawiasie wpisując tytuł okienka i jego treść. onclick="showModal('Tytuł', 'Treść')". Można używać elementów HTML w zawartości okienka. Naleleży jednak pamiętać, aby kod był pisany w jednej linii, a zmiast cudzysłowów używać &quot;. 

<button onclick="showModal('Tytuł modala', 'Treść modala')" class="btn">Modal</button>

<button onclick="showModal(
    '<p class = &quot; text-center &quot; style = &quot; width: 100%; &quot;>Promocja</p>', 
    '<p> <i> Tylko dziś: </i> kup 2kg pomarańczy w cenie 3kg. Taka okazja tylko dzisiaj i tylko u nas! </p> <p class = &quot; text-center &quot; style = &quot; width: 100%; &quot;> <button class=&quot;btn&quot;>Kup teraz</button> </p>'
    )" class="btn color-blue">Pokaż okienko</button>
