Komponenety

Aby przyspieszyć dostosowywanie powtarzalnych elementów strony można wykorzystać Komponenety. Komponenty to gotowe fragmenty kodu przechowywane w plikach w folderze /components.

header.html

<div class="header">
    <a href="/index.html">
        <div class="logo"></div>

        <h1>Website</h1>
    </a>
</div>

navigation.html

<nav class="top-nav">
    <ul>
        <li><a href="/index.html">Strona główna</a></li>
        <li>
            <a href="/options.html">Opcje</a>
            <ul>
                <li><a href="#">Fancy</a></li>
                <li><a href="#">Dropdown</a></li>
                <li><a href="#">Menu</a></li>
            </ul>
        </li>
        <li><a href="/about.html">O nas</a></li>
        <li><a href="/contact.html">Kontakt</a></li>
    </ul>
</nav>

menu.html

<div class="menu">
    <ul>
        <li> <a href="/category1.html"> <h1>Kategoria 1</h1> </a> </li>
        <li> <a href="#1">Podkategoria 1</a> </li>
        <li> <a href="#2">Podkategoria 2</a> </li>
        <li> <a href="#3">Podkategoria 3</a> </li>

        <li> <a href="/category2.html"> <h1>Kategoria 2</h1> </a> </li>
        <li> <a href="#1">Podkategoria 1</a> </li>
        <li> <a href="#2">Podkategoria 2</a> </li>
        <li> <a href="#3">Podkategoria 3</a> </li>
    </ul>
</div>

<div class="menu-toggle">☰ Menu</div>

footer.html

<div class="footer">
    <div class="row">
        <div class="col content-center"> Treść kolumny 1 stopki </div>
        <div class="col content-center"> Treść kolumny 2 stopki </div>
        <div class="col content-center"> Treść kolumny 3 stopki </div>
    </div>
</div>

Elementy

Implementacja Komponenetów na stronie jest bardzo prosta, wystarczy dodać odpowiedni znacznik.

html

<div id="header-placeholder"></div>

<div id="navigation-placeholder"></div>

<div id="menu-placeholder"></div>

...

<id="footer-placeholder">

Główną zawartość strony należy umieścić w elemencie div z klasą class="page-container". 

<div class="page-container">
    <!-- Tu główna zawartość strony -->
</div>

Pole z ramką

Pole z ramką może zostać wykorzystane m. in. do tworzenia artykułów lub formularzy. Dodawane jest za pomocą znacznika <div class="framed-field"></div> lub w skrócie <div class="frmdfld"></div>.
Można je przedzielić za pomocą poziomej linii: <hr class="divider-line"> lub w skrócie <hr class="divide">.

Układ elementów

Elementy można szybko układać za pomocą znaczników <div> z klasami "row" dla wiersza i "col" dla kolumny. Aby wyśrodkowąć zawartość elementu, należy dodać klasę "content-center".

class="row"
class="col"
class="col content-center" 
<div class="row">
    <div class="col"></div>
    <div class="col content-center"></div>
</div>
