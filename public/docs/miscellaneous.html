Różne

Ikony

Dla wizualnego urozmaicenia strony można dodać ikony. Jest to znacznik span z klasą "icon" oraz klasą wybranej ikony. Domyślnie zdefiniowane są poniższe ikony. 

<span class="icon play"></span>
<span class="icon layout"></span>
<span class="icon hand"></span>
<span class="icon text"></span>
<span class="icon brush"></span>

Aby utworzyć własne ikony, należy w pliku style.css zdefiniować klsę jako nazwę ikony oraz jako obraz tła podać ścieżkę do pliku z obrazkiem.

style.css 
.nazwa-ikony {background-image: url(/ścieżka/do/obrazka/ikony);}

Tabele

Tabele są tworzone według standardowych zasad HTML.

Lorem ipsum dolor sit 
<table>
    <tr>
        <th>Lorem</th> <th>ipsum</th> <th>dolor</th> <th>sit</th>
    </tr>
    <tr>
        <td rowspan="3"></td> <td></td> <td></td> <td></td>
    </tr>
    <tr>
        <td></td> <td></td> <td></td>
    </tr>
    <tr>
        <td></td> <td></td> <td></td>
    </tr>
    <tr>
        <td></td> <td></td> <td></td> <td></th>
    </tr>
</table>

Cień

Aby łatwo dodać do elementu cień, można dodać do niego klasę "shadow".

  Tekst z cieniem 
<button class="btn shadow">Guziol z cieniem</button>
<span class="shadow">Tekst z cieniem</span>

Powiększenie

Aby łatwo dodać do elementu powiększenie po najechaniu na niego, można dodać do niego klasę "hover-sizeup".

  Powiększenie 
<button class="btn hover-sizeup">Powiększenie</button>
<span class="lato-bold hover-sizeup">Powiększenie</span>
