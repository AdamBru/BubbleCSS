Czcionki

Domyślną czcionką użytą we frameworku jest Lato. Aby zastosować wybrany krój tej czcionki należy dodać odpowiednią klasę do elementu. Pełna lista klas jest dostępna w pliku fonts.css.

Lato Regular   Lato Bold   Lato Light Italic   Lato Thin
fragment pliku fonts.css 
/* ... */
.lato-regular {
    font-family: "Lato", sans-serif;
    font-weight: 400;
    font-style: normal;
}
/* ... */
.lato-bold {
    font-family: "Lato", sans-serif;
    font-weight: 700;
    font-style: normal;
}
/* ... */
.lato-light-italic {
    font-family: "Lato", sans-serif;
    font-weight: 300;
    font-style: italic;
}
/* ... */
.lato-thin {
    font-family: "Lato", sans-serif;
    font-weight: 100;
    font-style: normal;
}
/* ... */
html 
<span class="lato-regular">Lato Regular</span>
<span class="lato-bold">Lato Bold</span>
<span class="lato-light-italic">Lato Light Italic</span>
<span class="lato-thin">Lato Thin</span>

Rozmiar czcionki

Domyślny rozmiar czcionki to 20px. Aby dostosować rozmiar czcionki w danym elemencie zalecane jest używanie jednostki rem. Np. font-size: 1.5rem; to 150% wysokości domyśnego rozmiaru czcionki. Aby zmienić globaly domyślny rozmiar czcionki należy edytować plik config.css.

config.css 
:root {
    /* ... */
    font-size: 20px;
}

Listy

Lista nieuporządkowana <ul class="list">, lista uporządkowana <ol class="list"> i element listy <li> służą do tworzenia list. Aby oddzielić elementy listy linią należy dodać do listy klasę "list-underline".

<ol class="list">

<ol class="list list-underline">

<ul class="list">

<ul class="list list-underline">

  2. Element 2
  3. Element 3
  1. Element 1
  2. Element 2
  3. Element 3
  • Element 1
  • Element 2
  • Element 3
  • Element 1
  • Element 2
  • Element 3

    <li>Element 1</li>
    <li>Element 2</li>
